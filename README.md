# Interactive Maze Generator
Interactive Maze Generator, Solver & Game Engine (Python, Pygame, Randomised Prim's, A*)
Latest versions of python and pygame must be installed.
Keys: 
k - begin generating maze
1 - solve the maze
r - reset the maze grid
t - clear maze solution
2 - begin the game
q - place start at mouse pointer
w - place end at mouse pointer
n - use player ability 1
x - use player ability 2
mouse left click - toggle a maze wall at mouse position
mouse fight click - toggle a weighted cell at mouse position
